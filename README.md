# 🎯 AI Competency Profiler

AI-система для автоматического формирования компетентностного профиля специалиста на основе анализа вакансий.

Система извлекает требования из вакансий, декомпозирует их в иерархию компетенций и подкомпетенций, строит интерактивный граф знаний и позволяет эксперту валидировать и редактировать результат.

## 📋 Что делает

1. **Парсинг вакансии** — принимает ссылку на вакансию с hh.ru, извлекает описание и требования
2. **Определение категорий** — LLM автоматически определяет релевантные категории компетенций для любой профессии (не только IT)
3. **Извлечение требований** — структурированное извлечение скиллов из текста вакансии
4. **Декомпозиция компетенций** — каждый скилл раскрывается в иерархию подкомпетенций с уровнями (beginner / intermediate / advanced)
5. **Интерактивный граф** — визуализация в виде mind-map с возможностью редактирования
6. **Экспертная валидация** — эксперт может редактировать, удалять, добавлять компетенции вручную или с помощью AI

## 🏗️ Стек

| Компонент | Технологии |
|-----------|-----------|
| Frontend | React 18, Vite, Lucide Icons |
| Backend | FastAPI, Pydantic, BeautifulSoup |
| AI/LLM | OpenRouter API (DeepSeek, Llama, GPT) |
| Данные | hh.ru API |

## 🚀 Запуск

### Требования

- Python 3.10+
- Node.js 18+
- npm или yarn

### Backend

```
# Перейти в директорию бэкенда
cd backend

# Создать виртуальное окружение
python -m venv venv

# Активировать виртуальное окружение
# Linux / macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Запустить сервер
python main.py
```
Бэкенд запустится на http://localhost:8000

Swagger-документация: http://localhost:8000/docs

Frontend
```
# Перейти в директорию фронтенда
cd frontend

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```
Фронтенд запустится на http://localhost:5173

Vite автоматически проксирует запросы /api/* на бэкенд (порт 8000).
Убедитесь, что бэкенд запущен перед открытием фронтенда.

Быстрый старт (оба сервиса)
```
# Терминал 1
cd backend && python -m venv venv && source venv/bin/activate && pip install -r requirements.txt && python main.py

# Терминал 2
cd frontend && npm install && npm run dev

```
Открыть http://localhost:5173 → вставить ссылку на вакансию с hh.ru → готово.
